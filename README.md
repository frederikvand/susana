# SUSANA Project Data

This repository is a subset of the larger SUSANA project space, maintained specifically by the **Terrestrial Ecology Unit (TEREC)** at Ghent University.

**Main Goal:**
The primary objective of this folder is to develop a scientific manuscript (`evaluation/manuscript/SOM_dune_development_draft.tex`). This manuscript synthesizes findings from two core work package deliverables managed by UGent (Deliverable 1.1 and Deliverable 1.4) concerning boundary conditions for dune vegetation development and the fundamental/realised niche of marram grass on alternative sediments.

The empirical findings on biological dune development are combined with the geophysical storm erosion modeling conducted by KU Leuven, and importantly, with **dune development modelling using the Living Dunes model**. This will simulate dune restoration under conditions of normal versus alternative sediments (in total 10 different types). Together, these models will present a holistic ecosystem services evaluation of the dune-in-front-of-dike nature-based solution.

## Directory Structure

* **`data/`**: Project data inputs.
  * `geomorphology/`: point cloud scanes data.
  * `photos/`: Photographic records and imagery.
  * `substrate/`: Substrate and soil analysis data.
  * `vegetation_development/`: Data regarding plant growth and coverage.
  * `raw/`: Unorganized, raw data dumps.

* **`docs/`**: Project documentation and literature.
  * `dune_modelling/`: Documentation related to dune modeling approaches.
  * `literature/`: Reference papers and background literature.
  * `proposal/`: Grant and project proposals.
  * `protocol/`: Experimental protocols and methodologies.

* **`evaluation/`**: Analysis, results, and outputs meant for dissemination.
  * `analysis/`: R/Python/QGIS analysis projects and results.
  * `deliverables/`: Formal deliverables and reports (e.g., WP reports).
  * `manuscript/`: Drafts and final versions of the core scientific manuscript (`SOM_dune_development_draft.tex`).
  * `presentation/`: Slides, posters, and conference materials.

* **`scripts/`**: Source code for data processing and analysis.
  * `common_garden/`: Scripts for the common garden experiment data.
  * `germination/`: Scripts for analyzing germination experiment results.
  * `substrate/`: Scripts for substrate data analysis.

* **`output/`**: Generated figures, tables, and processed datasets from the `scripts/`.
  * `common_garden/`: Outputs from the common garden analysis.
  * `germination/`: Outputs from the germination analysis.
  * `substrate/`: Outputs from the substrate analysis.

* **`z_old/`**: Archived or deprecated files.
